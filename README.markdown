## GL Tanks
### A simple 3D tank game

Controls
--------
* WASD: Move and rotate tank body
* JL: Rotate turret
* Spacebar: Fire turret gun
* I: Speed boost (you loose some maneuverability while boosting)
* U: Radar sweep (allows you to see enemies and obstacles far away)
* O: Slow down time
* Escape: Quit
